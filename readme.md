# Gach-dong MSA

- Micro Service Architecture (Gateway Pattern)
- SpringBoot 3.0 + JDK 17
- Spring Data JPA
- Docker based Build & Deploy
- Prometheus and Grafana for monitoring
- Istio Ingress Gateway + Service Mesh
